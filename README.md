# CharGer
Characterization of Germline variants
## Install

	pip install .

## Configure
Add the following to your PATH environment (add it in ~/.bash_profile or ~/.bashrc)

	export PATH="<path to charger bin/>:$PATH"

## Run

	charger -i <variant file> -o <output file>

## For Help
Run:

	charger
